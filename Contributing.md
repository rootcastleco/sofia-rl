# Contributing to SofiaRL

Thank you for considering contributing to SofiaRL!

Please follow these guidelines:

- Fork the repository and create feature branches for changes.
- Write unit tests for any behavioral change.
- Follow code style (Black + isort). Run `pre-commit` hooks if available.
- Open a Pull Request with a clear description and link to related issues.
- For breaking changes or design proposals, open an issue first to discuss.

See `CODE_OF_CONDUCT.md` for community standards.
